# Pagination Builder Example

This example demonstrates how to use the Pagination Builder package in a Flutter application.

## Features Demonstrated

- Basic pagination with the factory constructor
- Advanced pagination with custom controller
- Custom data providers (API, database)
- State handling and UI customization
- Pull-to-refresh functionality

## Running the Example

```bash
cd example
flutter pub get
flutter run
```

The example app contains multiple screens showcasing different ways to use the pagination builder:

1. **Basic Example** - Simple list with automatic pagination
2. **Advanced Example** - Custom controller with more control options
3. **API Example** - Integration with a REST API
4. **State Handling** - Demonstration of different pagination states
5. **Database Example** - Integration with a local database

Each example includes detailed comments explaining the implementation details.
